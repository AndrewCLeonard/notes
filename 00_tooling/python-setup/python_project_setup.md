## Python Project Setup Ritual

### Once per machine

- Install Python from python.org **as administrator**
- Always use `python -m pip` instead of `pip` directly

### Once per project

1. `ni main.ipynb` — create your notebook
2. `py -m venv .venv` — create the venv
3. `.\.venv\Scripts\Activate.ps1` — activate it
4. `python -m pip install <packages>` — install dependencies

### Every time you return to a project

- Activate the venv before doing anything else
